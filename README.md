# The Apply Stack

Presentation / case study of three Claude Code skills Tim Zinin uses to automate job applications and B2B lead generation:

- **career-ops** — apply to VP Marketing / CMO roles on his behalf (fork of [santifer/career-ops](https://github.com/santifer/career-ops))
- **hh-outreach** — farm B2B leads via HH.ru vacancies ([github.com/TimmyZinin/hh-outreach](https://github.com/TimmyZinin/hh-outreach))
- **superjob-outreach** — same playbook for SuperJob.ru (in build)

Live: [timzinin.com/apply-stack](https://timzinin.com/apply-stack/)

Built with Claude Code (Opus 4.6). Static single-file site — no build, no tracking.
